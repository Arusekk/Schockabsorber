Section "Lscr"
--------------

Purpose of section::
Contains scripts (which are collections of functions).

Related sections::

- "LctX" -- Script collection
- "Lnam" -- Names used in scripts

### Format

    +========+       +================+    +===========+
    | Header |--+--->| Function table |--->| Functions |
    +========+  |    +================+    +===========+
                |\   +=========+
                | -->| Globals |
                |    +=========+
                |\   +============+
                | -->| Properties |
                |    +============+
                |\   +================+
                | -->| Handler vector |
                |    +================+
                |\   +==========+
                | -->| Literals |
                |    +==========+
                |          v
                 \   +================+
                  -->| Literal data   |
                     +================+
